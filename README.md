# 予防アプリ

予防医学のエビデンスをもとに、**性別・年齢・目的**に合わせた1週間の
**食事・運動・睡眠・休息・瞑想**を提案し、毎日の実践を記録・管理するWebアプリ（PWA）。

## 特徴
- 検査・専用器材なし。スーパーで買える食材／ネットで買えるサプリ／自宅でできる運動が中心
- 一人で・簡単に・続けられることを最優先
- サーバー不要。データは端末内（localStorage）に保存
- オフライン動作・ホーム画面に追加できるPWA

## 主な画面
ホーム（今日のプラン＋今日の学び） / 週間プラン / 目的別の献立 / 3分で学ぶコラム / 記録・推移 / 設定

## 使い方（ローカル）
```bash
cd app
python3 -m http.server 4173
# http://localhost:4173 を開く
```

## 構成
```
app/            公開されるWebアプリ本体（HTML/CSS/JS）
  js/cards.js     提案カード（食事/運動/睡眠/休息/瞑想/サプリ）
  js/meals.js     目的別の献立＋女性向けガイド
  js/articles.js  3分コラム
  ...
docs/           設計書
tools/          素材取り込みスクリプト（YouTube字幕 / Podcast文字起こし）
```

## 注意（公開範囲）
学習に用いた動画・ポッドキャストの文字起こしや書籍の抽出テキスト（`data/`）は
著作権の都合で本リポジトリには含めていません。`app/` 内のコンテンツは要約・再構成した独自テキストです。

> 医療アドバイスではありません。体調や持病がある場合は専門家にご相談ください。
